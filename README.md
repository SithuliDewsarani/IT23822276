# IT23822276
ITPM Playwright Assignment

Test cases are designed to verify whether the system correctly converts input text into the target language. They cover different scenarios such as sentence types, grammar variations, input lengths, punctuation, mixed language content, and informal expressions. Both successful and failure cases are included to identify system limitations and incorrect behavior.
Initialize node project- npm init -y
Install playwright- npm init playwright@latest
Run all Test cases - npx playwright test excel.spec.js
View HTML Test report - npx playwright show-report
